# Romulan Dictionary

Foreign/invented Romulan terms found in `examples/Star Trek Empires Pilot.fdx`.

| Romulan Term | English Term | Description |
|---|---|---|
| Riov | Commander / Captain | Romulan military rank for ship commanders — used for Talpek ("Algon Talpek, First Riov of the Vriha Thrai"), Dak'Len, and V'Ctuen. |
| Erei'riov | Sub-Commander | Title given to Thirul, second-in-command aboard the Vriha Thrai. |
| Khre'Arrain | (roughly) Sub-Commander / covert-operative rank | Revealed as Lt. McCoy's true rank/title: "Khre'Arrain Ri'ha Dannsu." |
| Erein | Ensign | Junior-officer address; Talpek addresses Ethnen: "Erein Ethnen, report." |
| Rekkhai | Sir / Commander | Respectful form of address to a superior officer; used constantly by subordinates addressing their Riov. |
| Au'e | Yes / Acknowledged / Affirmative | Standard Romulan affirmative response, used dozens of times throughout. |
| Jolan-Tru | Peace and long life (greeting/farewell) | Said by V'Ctuen and Talpek at partings; also said by Dak'Len after being dismissed. |
| Mnhei'sahe | "The Ruling Passion" | Romulan code of honor/duty; a SUPER title card translates Thirul's question "Mnhei'sahe?" directly. |
| Feanna | Fool! | Curse/exclamation shouted by Talpek when Dak'Len's ship is ambushed; translated on-screen via SUPER card. |
| Hna'h | Fire! / Energize! | Command to execute an action — used to order the shield lance fired and to order a planet's destruction. |
| Rhea Kveinn-Delevhas | Activate shield lance! | Talpek's command before firing the Vriha Thrai's new weapon on the Mnek'nra. |
| Praetorate | The ruling body/government of the Romulan Star Empire | Referenced repeatedly by Talpek and Thirul as the governing authority. |
| Lloan'na | Derogatory term for Federation/Starfleet personnel | Used repeatedly by Thirul, V'Ctuen, and a Romulan trooper as a slur for humans/outsiders. |
| Vriha Thrai | High-Predator | Name of the Romulan flagship; explicitly translated by Captain Irikov. |
| Mnek'nra | (ship name, untranslated) | Name of a Romulan troop transport ship. |
| Hfai | (ship name, untranslated) | V'Ctuen's Romulan troop-transport-turned-warship. |
| Lluadh Hfai | (ship name, untranslated) | Full/formal name of V'Ctuen's ship, a Romulan Valdore Class ship. |
| Valdore Class | Romulan starship class designation | Class of the Lluadh Hfai. |
| Donatra Seven | (tactical maneuver name) | Offensive deployment ordered by V'Ctuen during the Enterprise engagement. |
| Riov's Log | Captain's Log (Romulan equivalent) | Talpek's opening voiceover: "Riov's Log, Stardate 79113.2." |
| Romulan Star Empire | The Romulan polity's formal name | Used repeatedly throughout the script. |
| Romulan Ale | Iconic Romulan alcoholic beverage | Served at the officer's dinner; Thirul toasts "2490, a good year for Romulan Ale." |
